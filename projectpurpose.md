Purpose:
The Embassy Requirement Tracker is a chatbot-driven system designed to answer employee queries regarding visa processes and embassy requirements. It aims to reduce misinformation, improve efficiency for Sales and Operations teams, and centralize knowledge via a dashboard.

Scope:

Chatbot available via web/app for employees.

Dashboard for Admins to manage and update visa process information.

Phase 2: Automated data scraping from embassy websites to keep knowledge updated.



2. Functional Requirements

Chatbot Module:

Employees can ask questions about visa processes.

Bot responds with accurate, up-to-date information from the knowledge base.

Handles FAQs, step-by-step processes, required documents, and timelines.

Provides links/downloads for embassy forms if available.


Knowledge Dashboard:

Admin login (Sales Ops Managers, HR).

Add, edit, and delete visa requirement information.

Categorize data by country, visa type, embassy, etc.

Search and filter functionality for easy updates.


User Roles:

Employee (User): Can ask chatbot queries.

Admin (Manager/HR): Can manage content on dashboard.
